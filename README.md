# LeBonCoin Backend : 
REST API for leBonCoin reproduction project, built with Strapi.

## **Tech Stack**
- **CMS/API** : Strapi v4
- **Database** : SQLite (dev)
- **Image hosting** : Cloudinary

## **Features**
- user anthentication (JWT)
- Offers management (create, read, update)
- Custom route and policy for secure purchase flow
- Stripe webHook handling

 ## **Run locally**
 `npm install`
 
 `npm run develop`
